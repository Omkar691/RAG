python programing introduction
    Python is a high-level, interpreted programming language known for its simplicity and readability.
    It was created by Guido van Rossum and first released in 1991. Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming.
    It has a large standard library that provides tools suited to many tasks, making it a popular choice for web development, data analysis, artificial intelligence, scientific computing, and more. Python's syntax emphasizes code readability, allowing developers to express concepts in fewer lines of code compared to other languages. It is widely used in various fields due to its versatility and ease of learning.
